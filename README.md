# Jejak Nusantara

Versi ini sudah dipisah menjadi beberapa halaman:

- `home.html` — beranda / landing page
- `osn.html` — cerita OSN dan OSN AI
- `pelatnas.html` — cerita Pelatnas IOI dan Pelatnas IOAI
- `edutrip.html` — cerita Museum MACAN, Kota Tua, Tzu Chi, dan SAAJA
- `pendukung.html` — transportasi, akomodasi, makanan, dan dokumen
- `index.html` — login lokal

Perbaikan terbaru:

- Search bar di header dikembalikan menjadi search bar asli.
- Tombol OSN, Pelatnas, Edutrip, dan Pendukung tetap ada di navbar, bukan menggantikan search.
- Foto tidak lagi ditampilkan sebagai grid besar yang langsung menumpuk semuanya.
- Foto utama ditemani 1–2 foto pendukung, sedangkan foto lain disimpan dalam bagian “Lihat foto/video lainnya”.
- Setiap halaman memakai pola layout yang berbeda supaya tidak terasa template yang sama terus.
- Teks Pelatnas dibuat lebih panjang, lebih personal, dan lebih manusiawi, terutama untuk Pelatnas IOAI tahap 1 dan tahap 2.
- Bullet point pada halaman detail diganti menjadi paragraf naratif yang lebih enak dibaca.

Akun masih memakai mode lokal di browser dan tidak memakai Supabase.


## Update layout
Foto dan video sekarang tidak hanya dikumpulkan di popup. Pada halaman detail, dokumentasi disebar di tengah paragraf sebagai editorial media flow: beberapa paragraf, lalu beberapa foto yang relevan, lalu cerita berlanjut.
